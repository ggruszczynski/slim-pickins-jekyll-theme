test Slim Pickins is what I consider to be a minimal 'core' theme with everything I need to quickly develop new themes for Jekyll. Here is an example MathJax inline rendering \\( 1/x^{2} \\), and here is a block rendering: \\[ \frac{1}{n^{2}} \\]

Basic features include:

  • Jekyll 3.0 Compatibility
  • SASS
  • Minimal Foundation Zurb 6 integration
  • Responsive grid
  • Responsive navigation
  • Optional full-width banner
  • Sticky footer
  • Javascript free
  • Custom Rakefile with tasks for deploying and notifying search engines about updates
  • image_optim plugin to optimize all images
  • Basic SEO
